### 프로젝트 소개

`Gongspot`은 청년들의 공부 공간 정보(혼잡도, 리뷰)등을 공유할 수 있는 서비스입니다. **SpringBoot**를 기반으로 개발되었으며, RESTful API를 통해 클라이언트와 통신합니다.

### 기술 스택

- **Backend**: Spring Boot, Spring Security, JPA, MySQL
- **Deployment**: AWS EC2
